# VRPSolver
Ejecución del código 
Consideraciones previas a ejecución: 
● Se debe tener el problema que se quiera ejecutar en la carpeta /sets del proyecto. (ya incluye una gran cantidad de instancias para llegar y probar, pero por si acaso, revisar la lista)
● Este proyecto fue desarrollado en C++ versión 17, con el compilador G++ versión 13. 
Instrucciones de ejecución: 
● Con la carpeta descomprimida y abierta en una terminal (de linux o VSC), se debe de ejecutar la instrucción ‘make main’ sin las comillas. Esta instrucción creará todos los archivos necesarios para ejecutar el main. Si se quiere ejecutar también los tests, para crearlos se deberá ejecutar la instrucción ‘make all’ sin las comillas. 
● Para ejecutar el main, tras haberlo creado exitosamente, se debe ejecutar con la instrucción ‘./main’ sin las comillas. Para cada test individual, se debe replicar esto. pero con el nombre del test después de ‘./’ (ej: ./test_alns) 
● Tras ejecutar el main, se deben seguir las instrucciones del menú que aparecerán en pantalla para hacer lo que se desee (cargar un archivo, ejecutar un algoritmo, etc) 
● Si se quiere finalizar la ejecución del programa, se debe elegir la opción ‘salir’ del menú principal, la cual abortará la ejecución.
